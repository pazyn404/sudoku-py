# Sudoku project

## Installation

* Windows:
```
git clone https://github.com/pazyn404/sudoku-py.git
cd sudoku-py
python -m venv venv
venv\Scripts\Activate
python -m pip install -r requirements.txt
```

* Linux, MacOS
```
git clone https://github.com/pazyn404/sudoku-py.git
cd sudoku-py
python -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
```
